Tutorials¶

This section covers tutorials that will explain step by step approach to build GStreamer pipelines for different usecases using VVAS framework. Each tutorial explains what is the purpose of each plugin in the pipeline and how to configure, set properties and the corresponding json files whereever applicable.

Multi Channel ML¶

This tutorial explains how to build Machine Learning pipelines using VVAS. This tutorial first covers single stage Machine Learning Inference pipeline where only one ML operation is required on input image. Complex Real life applications may require more than one stages of Machine Learning operations to get the required information. When more than once Machine Learning Inference operations are performed, then this is commonly known as Cascaded Machine Learning. This tutorial also covers how to build multi stage Cascaded Machine Learning Pipelines using VVAS. For more details, refer to the link MultiChannelML .
